云服务器指北-拥有属于一台自己的服务器

2022-08-24 Comments

购置云服务器

云服务器连接至本地

每次登陆云平台比较繁琐，配置远程连接，本地可以登录

首先初始化root用户

sudo passwd #初始化密码

su #进入root用户

修改配置文件

vim /etc/ssh/sshd_config

在其中设置

PubkeyAuthentication yes
RSAAuthentication yes
AuthorizedKeysFile 取消注释
PasswordAuthentication yes

修改后重启

service sshd restart

添加用户

adduser wzm

设置密码并且添加相关信息

添加sudo权限

vim /etc/sudoers

仿照root权限 添加用户的权限

在本地连接服务器

ssh <用户>@<ip>

为本地服务器配置公钥，以便每次链接时跳过输入密码的环节

在本地terminal输入

ssh-keygen -t rsa

在本地 C:/用户/用户名/.ssh文件夹下打开id_rsa.pub文件，全部复制

在服务器用户根目录下

mkdir .ssh
cd .ssh
touch authorized_keys
vim authorized_keys

写入刚才复制的公钥

重启服务

sudo service sshd restart

现在本地不输入密码也可以登录

为云服务器配置数据库

进入服务器，输入

sudo apt install mysql-server

安装mysql

检查服务状态

sudo systemctl status mysql

修改连接配置使得数据库可被其他ip访问

sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf

修改

bind-address      = 0.0.0.0

重启服务

sudo systemctl restart mysql

进入mysql服务

sudo mysql
